# gcluster
go game server

# star it && QQ群：869428810

# Features
1. 基于ioc容器
2. 便捷的使用、扩展、配置
3. http/tcp/websocket网络支持
4. redis/mongo数据库支持
5. 数据的自动同步处理，客户端的增量更新，和数据的自动同步
